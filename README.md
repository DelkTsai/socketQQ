使用前提: 电脑装有JDK,即有java虚拟环境

双击Server.jar(或Server.bat),前者无效试试后者,然后跳出一个窗口,点击开启服务器.

然后双击Client.jar(或Client.bat),前者无效试试后者,然后跳出一个窗口,输入ID(任意字符,但不能重复)即可登录.

本程序仅仅能够在本机实现多客户端聊天!若需实现局域网聊天,自己改代码即可(大概找到GGLogin.java,改掉Socket的连接主机参数,改成开了服务器的那个电脑的局域网IP(如192.168.1.X))

实现:一人同时与多人聊天(不是群聊)
    任何人(默认为任何连接服务器的连接都为好友,自己想办法可以修改成按好友)  上线都会提示!(弹出提示框)

Tips:测试聊天时,开启多个客户端(Client.jar),相互之间可以聊天!
